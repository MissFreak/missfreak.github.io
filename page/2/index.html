Attention模型中注意力的多尺度视觉化

发表于 2021-08-01 更新于 2021-08-07 分类于 项目 ， 信息抽取
本文字数： 1.2k 阅读时长 ≈ 1 分钟

Transformer是一个序列模型，它放弃了传统的递归架构，而采用了完全基于注意力的方法。除了提高性能外，使用注意力的一个好处是，它还可以通过显示模型如何为不同的输入元素分配权重来帮助解释一个模型。然而，Transformer模型中的多层、多头注意力机制可能难以解读。为了使该模型更容易理解，我们介绍了一个开源工具，该工具在多个尺度上将注意力可视化，每个尺度都为注意力机制提供一个独特的视角。我们在BERT和OpenAI GPT-2上演示了该工具，并提出了三个用例：检测模型的偏差，定位相关的注意力头，以及将神经元与模型行为联系起来。

一袋什么？用于文本分析的简单名词短语抽取

发表于 2021-07-28 更新于 2021-08-11 分类于 项目 ， 信息抽取
本文字数： 7.6k 阅读时长 ≈ 7 分钟

没有经过专门的自然语言处理培训的社会科学家在分析文本语料库时经常使用unigram词袋（BOW）表示。我们提供了一种新的基于短语的方法，即NPFST，用于丰富unigram词袋的内容。NPFST使用一个POS标记器和一个有限状态转换器来提取多词短语，并将其添加到unigram词袋中。我们将NPFST与Ngram和解析方法在产量（yield）、召回率（recall）和效率（efficiency）方面进行比较。然后我们演示了如何使用NPFST进行探索性分析；它在许多不同种类的英语文本上表现良好，无需配置。最后，我们介绍了一个使用NPFST分析美国国会法案(congressional bills ) 新语料库的案例研究。关于我们的开源实现，见http://slanglab.cs.umass.edu/phrases/。
